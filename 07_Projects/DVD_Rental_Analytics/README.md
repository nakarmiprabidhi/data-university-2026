# DVD Rental Business Analytics

## Project Overview

This project analyzes the operations, customers, rentals, inventory, and revenue of a fictional DVD rental company.

The objective is to transform transactional data stored in PostgreSQL into business insights, analytical reports, and interactive dashboards.

## Business Objectives

The project will answer questions related to:

- Customer activity and retention
- Store performance
- Rental behavior
- Film and category popularity
- Inventory utilization
- Revenue performance
- Monthly business trends

## Technologies

- PostgreSQL
- SQL
- Python
- Pandas
- Power BI
- Tableau
- Git and GitHub
- AWS concepts

## Project Workflow

1. Explore the PostgreSQL database.
2. Write and validate SQL queries.
3. Extract data using Python.
4. Clean and analyze data with Pandas.
5. Build Power BI and Tableau dashboards.
6. Design an AWS-based analytics architecture.
7. Document business findings and recommendations.
